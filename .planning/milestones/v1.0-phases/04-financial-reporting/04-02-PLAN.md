**creditor.service.ts** — 8 exported functions:

1. `createCreditor(input: CreateCreditorInput, actorId: string)`: Effect that inserts into creditors table, writes audit log
2. `updateCreditor(id: string, input: UpdateCreditorInput, actorId: string)`: Effect that updates creditor, writes audit log
3. `getCreditor(id: string)`: Effect that fetches creditor by ID or returns CreditorNotFound
4. `listCreditors()`: Effect that returns all creditors ordered by name
5. `addInvestment(input: AddInvestmentInput, actorId: string)`: Effect that inserts into creditor_investments with principalBalance = amount, writes audit log
6. `recordCreditorRepayment(input: RecordCreditorRepaymentInput, actorId: string)`: Effect inside db.transaction() that:
   - Fetches investment, computes interest accrued since last repayment using `calculateInterest(principalBalance, rate, daysElapsed, 0)`
   - Uses `allocatePayment({...params, minInterestDays: 0})` for allocation
   - Inserts creditor_repayments row with interestPortion, principalPortion, balanceBefore/After
   - Updates investment principalBalance
   - Writes audit log inside tx
7. `getCreditorDashboard(creditorId: string)`: Effect that computes all KPIs per-investment using engine.ts functions
8. `getSystemCapital()`: Effect that aggregates across all creditors for CRED-06

**actions.ts** — Server Actions (per project convention, no Zod):

- `addInvestmentAction(formData)` — extracts investment fields, calls addInvestment
