    **Step 2: Create src/services/daily-collections.service.ts**

    Two exported Effect functions:

    `getDailyCollections(date: string)` — accepts YYYY-MM-DD string:
    - Query: SELECT paymentId, loanId, customerName, amount, interestPortion, principalPortion, paymentDate FROM payments INNER JOIN loans ON payments.loanId = loans.id INNER JOIN customers ON loans.customerId = customers.id
    - WHERE: `isNull(payments.deletedAt)` AND `sql\`DATE(${payments.paymentDate} AT TIME ZONE 'Africa/Kampala') = ${date}::date\``
    - ORDER BY: `payments.paymentDate` ascending
    - Aggregate totalCollected using BigNumber: `rows.reduce((s, r) => s.plus(new BigNumber(r.amount)), new BigNumber(0)).toFixed(2)`
    - Return `{ date, totalCollected, paymentCount: rows.length, rows }`
    - Wrap in `Effect.tryPromise({ try: async () => {...}, catch: (e) => new DatabaseError({ cause: e }) })`

    `getLoansDueToday()` — no args:
    - Query all active loans: `db.select().from(loans).where(and(eq(loans.status, "active"), isNull(loans.deletedAt)))`
    - For each loan, query its non-deleted payments ordered by paymentDate ASC
    - Compute daysSinceLastPayment: if loan has payments, use last payment's paymentDate; else use loan.startDate. Formula: `Math.floor((now.getTime() - lastDate.getTime()) / (1000 * 60 * 60 * 24))`
    - If daysSinceLastPayment >= 30, fetch customer name, compute outstandingBalance from lastPayment.principalBalanceAfter (or loan.principalAmount if no payments)
    - Sort results by daysSinceLastPayment descending
    - Wrap in Effect.tryPromise same as above

    Import pattern: `import { Effect } from "effect"`, `import { db } from "@/lib/db"`, `import { payments } from "@/lib/db/schema/payments"`, `import { loans } from "@/lib/db/schema/loans"`, `import { customers } from "@/lib/db/schema/customers"`, `import { sql, eq, and, isNull, asc } from "drizzle-orm"`, `import { DatabaseError } from "@/lib/errors"`, `import BigNumber from "bignumber.js"`, `import type { DailyCollectionsSummary, LoanDueToday } from "@/types"`
