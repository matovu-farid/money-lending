  <action>
    Create `src/components/ui/drawer-dialog.tsx` as a "use client" component. This file exports two components: `DrawerDialog` and `DrawerDialogContent`.

    **DrawerDialog (root wrapper):**
    - Props: `{ open: boolean; onOpenChange: (open: boolean) => void; children: React.ReactNode }`
    - Uses `useMediaQuery("(min-width: 768px)", { defaultMatches: true })` from `@base-ui/react/unstable-use-media-query` to detect desktop
    - Desktop path: renders `<Dialog open={open} onOpenChange={onOpenChange}>{children}</Dialog>` using the existing Dialog from `@/components/ui/dialog`
    - Mobile path: renders `<Drawer.Root open={open} onOpenChange={(o) => onOpenChange(o)} swipeDirection="down">{children}</Drawer.Root>` using `Drawer` from `@base-ui/react/drawer`
    - The `onOpenChange` wrapper `(o) => onOpenChange(o)` normalizes the Drawer's two-argument callback to single-argument (per RESEARCH.md Pitfall 2)

    **DrawerDialogContent (content wrapper):**
    - Props: `{ className?: string; children: React.ReactNode; showCloseButton?: boolean }` plus spread `...props`
    - Uses same `useMediaQuery` call as DrawerDialog (each component calls it independently — React hooks are cheap and this avoids context threading)
    - Desktop path: renders `<DialogContent className={className} showCloseButton={showCloseButton} {...props}>{children}</DialogContent>`
    - Mobile path: renders:
      ```
      <Drawer.Portal>
        <Drawer.Backdrop className="fixed inset-0 z-50 bg-black/10 supports-backdrop-filter:backdrop-blur-[24px]" />
        <Drawer.Popup
          data-slot="drawer-dialog-content"
          className={cn(
            "fixed bottom-0 inset-x-0 z-50 flex flex-col bg-white/85 rounded-t-xl p-4 max-h-[90dvh] overflow-y-auto",
            "data-open:animate-in data-open:slide-in-from-bottom",
            "data-closed:animate-out data-closed:slide-out-to-bottom",
            className
          )}
          {...props}
        >
          {/* Drag handle */}
          <div className="mx-auto mb-4 h-2 w-12 rounded-full bg-muted-foreground/30" />
          {children}
          {showCloseButton !== false && (
            <Drawer.Close
              render={
                <Button
                  variant="ghost"
                  className="absolute top-2 right-2"
                  size="icon-sm"
                  aria-label="Close"
                />
              }
            >
              <XIcon />
              <span className="sr-only">Close</span>
            </Drawer.Close>
          )}
        </Drawer.Popup>
      </Drawer.Portal>
      ```

    **Imports needed:**
    ```typescript
    "use client"

    import * as React from "react"
    import { useMediaQuery } from "@base-ui/react/unstable-use-media-query"
    import { Drawer } from "@base-ui/react/drawer"

    import {
      Dialog,

    } from "@/components/ui/dialog"
    import { Button } from "@/components/ui/button"
    import { cn } from "@/lib/utils"
    ```
