<!-- From 01-RESEARCH.md Pattern 6: Interest Engine -->
The Interest Engine is a set of pure functions with NO side effects, NO database calls.
Formula: interest = outstanding_balance x daily_rate x days_elapsed
Daily rate = monthly_rate / 30
Minimum interest period: max(daysElapsed, minInterestDays)
BigNumber.config({ DECIMAL_PLACES: 10, ROUNDING_MODE: BigNumber.ROUND_HALF_UP })
formatAmount uses .toFixed(2) for display/storage

From 01-CONTEXT.md Loan Ledger Model:
- Loans are perpetual -- NO termDays, NO dueDate
- Daily rate formula: daily_rate = current_principal x monthly_rate / 30
- Days overdue formula: days_overdue = unpaid_interest / current_daily_rate
- Flag when days_overdue >= 30

From 01-CONTEXT.md Interest Calculation Preview (Review Step):
- Daily interest amount (UGX)
- Total interest at minimum period
- Total owed at minimum period (principal + interest)
- Minimum interest period reminder
