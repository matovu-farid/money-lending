## Spacing Scale

Declared values (all multiples of 4, pulled from existing Tailwind v4 + shadcn token usage):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-label gap inside tab buttons (`gap-1` = 4px) |
| sm | 8px | Compact padding within MoreSheet nav items (horizontal inset) |
| md | 16px | Page-level padding (mobile), MoreSheet interior `p-4` |
| lg | 24px | Page-level padding (desktop md+), `md:p-6` |
| xl | 32px | Not used in this phase |
| 2xl | 48px | Not used in this phase |
| 3xl | 64px | Not used in this phase |

Exceptions:
- BottomTabBar height: **56px** (`h-14`) — matches TopBar height for visual symmetry
- BottomTabBar safe-area padding: `pb-[env(safe-area-inset-bottom)]` on the `<nav>` element (can be 0px on non-notched devices, ~34px on iPhone with Face ID)
- Main content bottom padding on mobile: `pb-[calc(3.5rem+env(safe-area-inset-bottom))]` = `pb-[calc(56px+env(safe-area-inset-bottom))]` — prevents fixed bar from obscuring content
- MoreSheet drag handle: `h-1 w-10` (4px × 40px) centered at top of popup, `mt-2`
- Touch targets: all tab buttons and MoreSheet nav items meet 44px minimum height (WCAG 2.5.8) via `h-14` on tabs and `py-3` on MoreSheet items

## Color

All values use CSS custom properties from `src/app/globals.css` (Sovereign Ledger palette). Do not use hardcoded hex.

| Role | CSS Variable | Light Value | Usage |
|------|-------------|-------------|-------|
| Dominant (60%) | `--background` | oklch(0.98 0 0) ≈ #f9f9fb | Page surfaces, TopBar, BottomTabBar background |
| Secondary (30%) | `--sidebar` | oklch(0.96 0 0) ≈ #f3f3f5 | MoreSheet popup background (same as sidebar) |
| Accent (10%) | `--sidebar-accent` | oklch(0.94 0 0) ≈ #edeef0 | Active MoreSheet items hover/active bg |
| Active indicator | `--primary` | oklch(0 0 0) = #000000 | Active tab text color + active indicator line |
| Inactive tab | `--muted-foreground` | oklch(0.42 0 0) ≈ #474747 | Inactive tab icon + label color |
| Border | `--border` | oklch(0.8 0 0) ≈ #c6c6c6 | BottomTabBar top border (`border-t border-border`) |
| Sheet backdrop | black/20 | rgba(0,0,0,0.20) | MoreSheet `Drawer.Backdrop` overlay |
| Destructive | `--destructive` | oklch(0.577 0.245 27.325) | Not used in this phase |

Accent reserved for: active MoreSheet nav item background (`bg-sidebar-accent`) and active MoreSheet nav item hover state. **Not** used on BottomTabBar — the bottom tab uses `text-primary` (black) for the active state, not a background highlight.

Active state indicator line: 2px tall (`h-0.5`), 32px wide (`w-8`), rounded (`rounded-full`), `bg-primary`, positioned absolute at bottom of the tab button, opacity transitions between 0 and 1.

## Tab Bar Specification

### Primary Tabs (BottomTabBar — visible only at < md viewport)

| Position | Label | Icon | Route | Active Detection |
|----------|-------|------|-------|-----------------|
| 1 | Dashboard | `LayoutDashboard` (h-5 w-5) | `/dashboard` | `pathname === '/dashboard' \|\| pathname.startsWith('/dashboard/')` |
| 2 | Customers | `Users` (h-5 w-5) | `/customers` | `pathname === '/customers' \|\| pathname.startsWith('/customers/')` |
| 3 | Payments | `CreditCard` (h-5 w-5) | `/payments` | `pathname === '/payments' \|\| pathname.startsWith('/payments/')` |
| 4 | Loans | `Banknote` (h-5 w-5) | `/loans` | `pathname === '/loans' \|\| pathname.startsWith('/loans/')` |
| 5 | More | `MoreHorizontal` (h-5 w-5) | none — opens MoreSheet | never active (always `text-muted-foreground`) |

Layout: `flex h-14 items-stretch`. Each tab: `flex-1 flex flex-col items-center justify-center gap-1`.

### Secondary Items (MoreSheet — @base-ui/react Drawer, swipeDirection="down")

| Position | Label | Icon | Route |
|----------|-------|------|-------|
| 1 | Creditors | `Landmark` (h-5 w-5) | `/creditors` |
| 2 | Expenses | `Receipt` (h-5 w-5) | `/expenses` |
| 3 | Income | `TrendingUp` (h-5 w-5) | `/income` |
| 4 | Reports | `BarChart3` (h-5 w-5) | `/reports` |
| 5 | Watchlist | `AlertTriangle` (h-5 w-5) | `/watchlist` |

Note: `/income` is confirmed as a separate route (`src/app/(app)/income/` exists). The sidebar currently shows "Expenses & Income" as one item linking to `/expenses`; the MoreSheet splits these into two separate links per REQUIREMENTS.md NAV-02.

MoreSheet popup: `fixed bottom-0 inset-x-0 z-50 bg-background rounded-t-2xl pb-[env(safe-area-inset-bottom)]`. Nav items: `rounded-md px-3 py-3 text-sm` — active uses `bg-sidebar-accent text-sidebar-accent-foreground font-medium`, inactive uses `text-foreground hover:bg-accent`. Clicking a MoreSheet item closes the sheet.

## Visibility Rules (CSS-only — no JS viewport detection)

| Element | Mobile (< 768px) | Tablet/Desktop (768px+) |
|---------|-----------------|------------------------|
| BottomTabBar | `flex` (visible) | `hidden` (absent from layout flow) |
| Sidebar | `hidden` (absent from layout flow) | `flex` (visible) |
| TopBar hamburger button | REMOVED this phase | REMOVED this phase |
| Sheet mobile sidebar | REMOVED this phase | n/a |

Implementation: `<BottomTabBar className="flex md:hidden" />` and `<div className="hidden md:flex"><Sidebar /></div>`. Use `flex md:hidden` / `hidden md:flex` — never `visibility:hidden` or `opacity:0` (those keep elements in layout flow).

## Interaction Contract

| Interaction | Behavior |
|------------|----------|
| Tap primary tab (not active) | Navigate to tab's route; active indicator transitions to new tab via `transition-colors duration-200` + `transition-opacity duration-200` |
| Tap primary tab (already active) | No navigation (Link behavior — same URL push is a no-op in Next.js) |
| Tap "More" tab | Opens MoreSheet via `useState(moreOpen)` in AppShell; sheet slides up from bottom |
| Swipe down on MoreSheet | Dismisses sheet (handled by @base-ui/react Drawer `swipeDirection="down"`) |
| Tap MoreSheet item | Navigates to route; calls `onOpenChange(false)` to close sheet |
| Tap MoreSheet backdrop | Dismisses sheet (Drawer.Backdrop click closes) |
| Resize to tablet+ | CSS-only: BottomTabBar removed from layout flow; Sidebar appears — no JS event needed |
