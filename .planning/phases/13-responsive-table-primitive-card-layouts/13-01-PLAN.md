Define and export these types:

```typescript
export type Column<T> = {
  key: string
  header: string
  render: (row: T) => React.ReactNode
  primary?: boolean
  hideInCard?: boolean
  align?: "left" | "right"
  cardLabel?: string
}

export type ResponsiveTableProps<T> = {
  columns: Column<T>[]
  rows: T[]
  getRowKey: (row: T) => string
  getRowProps?: (row: T) => React.HTMLAttributes<HTMLElement>
  emptyState?: React.ReactNode
}
```

Export a generic `ResponsiveTable<T>` function component:

1. If `rows.length === 0 && emptyState`, return `<>{emptyState}</>`.

2. Render TWO sibling blocks:

**Desktop table** (`hidden md:block` wrapper div):
- Wrap the existing shadcn `<Table>` inside `<div className="hidden md:block">`.
- `<TableHeader>` with `<TableRow>` mapping `columns` to `<TableHead>`. Apply `className="text-right"` when `col.align === "right"`.
- `<TableBody>` mapping `rows`. Each `<TableRow>` receives `key={getRowKey(row)}` and spreads `{...getRowProps?.(row)}`. Each cell maps columns to `<TableCell>` with `className="text-right"` when `col.align === "right"`, rendering `col.render(row)`.

**Mobile card list** (`md:hidden` wrapper div):
- `<div className="md:hidden space-y-2">`.
- Map `rows`. Each card is a `<div>` with:
  - `key={getRowKey(row)}`
  - `data-testid="data-row"` (hardcoded on every card for Cypress compatibility)
  - `className="rounded-lg border bg-card p-4 space-y-2"` plus conditionally add `cursor-pointer hover:bg-muted/50 transition-colors` when `getRowProps` returns an `onClick`.
  - Forward `onClick`, `role`, `aria-label`, `style` from `getRowProps?.(row)` onto the card div.

Inside each card:
- **Primary row**: Find `columns.find(c => c.primary) ?? columns[0]`. Determine if there's an actions column: `const actionsCol = columns.find(c => c.key === "actions")`.
  - If actionsCol exists: `<div className="flex justify-between items-center">` with `<div className="flex-1 text-sm font-semibold">{primaryCol.render(row)}</div>` and `<div>{actionsCol.render(row)}</div>`.
  - If no actionsCol: `<div className="text-sm font-semibold">{primaryCol.render(row)}</div>`.

- **Detail grid**: Filter columns to exclude primary, hideInCard=true, and key="actions". Map remaining to a `<dl className="grid grid-cols-2 gap-x-4 gap-y-1 text-sm">` with:
  - `<dt className="text-muted-foreground">{col.cardLabel ?? col.header}</dt>`
  - `<dd className={col.align === "right" ? "text-right font-mono tabular-nums" : undefined}>{col.render(row)}</dd>`
  - Wrap each dt/dd pair in a `<React.Fragment key={col.key}>`.

Import Table/TableHeader/TableBody/TableRow/TableHead/TableCell from `@/components/ui/table`.
